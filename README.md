# Visual Summary Report — Task 4

## Dataset: E-commerce Sales Data
## Tool Used: Python (Matplotlib, Seaborn)

## Charts Created:

### 1. Bar Chart — Monthly Sales
- Shows sales for each month
- November & December have highest sales (festive season)

### 2. Line Chart — Sales & Profit Trend
- Both sales and profit grow steadily
- Q4 (Oct–Dec) shows the strongest growth

### 3. Pie Chart — Sales by Category
- Electronics dominates with ~55% share
- Clothing is second, Furniture is third

### 4. Bar Chart — Sales by Region
- Sales are fairly distributed across regions
- West and South regions lead slightly

### 5. Dashboard — All 4 Charts Combined
- Single view of all key metrics

## Interview Answers:

### Best chart for trends?
→ Line Chart is best for showing trends over time
  because it clearly shows increase/decrease patterns.

### Dashboard vs Report?
→ Dashboard = real-time, visual, interactive summary
  of KPIs for quick decision making.
→ Report = detailed, static document with in-depth
  analysis, tables, and written explanations.
